# 🚀 JobMatch Pro – AI-Powered Job Recommendation System

JobMatch Pro is a full-stack web application that analyzes user CVs and provides **personalized job recommendations** using intelligent skill extraction and matching algorithms.

---

## ✨ Key Features

* 🔐 Secure user authentication (Gmail + strong password validation)
* 📄 AI-based CV parsing & skill extraction
* 🎯 Personalized job recommendations
* 🔍 Job search with CV-job match analysis
* 📊 Application tracking dashboard
* 👤 Complete job profile builder (skills, projects, experience)
* 🌙 Light/Dark mode with smooth UI animations

---

## 🛠️ Tech Stack

**Frontend:** HTML, CSS, JavaScript
**Backend:** Python (Flask)
**Database:** SQLite
**Libraries:** PyMuPDF, pdfplumber, PyPDF2, python-docx

---

## ⚡ Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run backend
cd backend
python app.py
```

Open in browser:
👉 http://localhost:5000
or open `frontend/index.html`

---

## 📌 How It Works

1. User uploads CV
2. System extracts skills using NLP-based parsing
3. Matches skills with job dataset
4. Generates personalized job recommendations with match score
